## 📋 Overview

**Z-Campus** is a mobile application developed with Flutter, aimed at providing practical and accessible solutions for campus-related activities. The project focuses on clean UI structure, responsive design, and providing an intuitive experience for end-users.

## ✨ Features

* **Cross-Platform Compatibility:** Runs smoothly on both iOS and Android.
* **Modern UI:** Built using Material Design principles and custom widgets for a sleek look.
* **State Management:** Efficient and predictable state management using Dart's advanced patterns.
* **Optimized Performance:** Fast load times and smooth scrolling experiences.

## 🚀 Getting Started

### Prerequisites

* Flutter SDK `3.x`
* Dart SDK `3.x`

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tahaemree/Z-Campus.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Z-Campus
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
